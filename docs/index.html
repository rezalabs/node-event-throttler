High-Velocity
Event Tracking.

An aggregation engine for Node.js designed to handle high-velocity event streams through atomic deferral and batching logic.

View Implementation
$ npm install node-event-throttler 
const EventTracker = require('node-event-throttler');

const tracker = new EventTracker({
  limit: 50,
  deferInterval: 30000,
  processor: async (batch) => {
    // Atomic batch processing
    await db.save(batch);
  }
});
CORE ARCHITECTURE

Atomic Event Deferral

Protect downstream services by enforcing precise rate limits. When thresholds are exceeded, events are automatically aggregated for scheduled batch processing.

// Seamless event ingestion
const result = await tracker.trackEvent('auth', 'uid_01', {
  ip: '127.0.0.1',
  action: 'login'
});
// result.type: 'immediate' | 'deferred'
PERSISTENCE STRATEGY

Distributed Redis State

Maintain consistent event tracking across multiple Node.js instances. The Redis adapter ensures atomic operations and data integrity in distributed environments.

const { createClient } = require('redis');
const EventTracker = require('node-event-throttler');
const { RedisAdapter } = EventTracker;
const { scripts } = RedisAdapter;

const redisClient = createClient({ 
  url: 'redis://localhost:6379',
  scripts 
});

await redisClient.connect();

const tracker = new EventTracker({
  storage: new RedisAdapter({ redisClient }),
  limit: 100,
  deferInterval: 60000
});
ALGORITHMIC FLEXIBILITY

Token Bucket Bursts

Allow short bursts of traffic while enforcing a strict long-term average rate. Use the Token Bucket strategy to handle variable load without compromising stability.

const EventTracker = require('node-event-throttler');
const { TokenBucketStrategy } = EventTracker;

const tracker = new EventTracker({
  strategy: new TokenBucketStrategy({
    bucketSize: 20,
    refillRate: 1 // tokens per second
  })
});

Robust Error Handling

Integrate with Dead Letter Queues or retry mechanisms through a reliable event-driven lifecycle.

tracker.on('process_failed', ({ error, events }) => {
  // Access failed batch for recovery
  console.error(error.message);
  await dlq.push(events);
});
